# 关于加强境内企业境外发行证券和上市相关保密和档案管理工作的规定

## 基本信息

- **发布机构**：中国证券监督管理委员会、财政部、国家保密局、国家档案局
- **发布日期**：2023年2月24日
- **生效日期**：2023年3月31日
- **文号**：证监会公告〔2023〕44号
- **效力状态**：✅ 现行有效

## 核心要点

### 一、适用范围

| 主体类型 | 说明 |
|---------|------|
| 境内企业 | 包括直接境外发行上市的境内股份有限公司和间接境外发行上市主体的境内运营实体 |
| 证券公司、证券服务机构 | 从事境内企业境外发行上市业务的境内外证券公司、证券服务机构 |

### 二、保密要求

#### 涉及国家秘密、国家机关工作秘密的文件资料

**提供条件**：
1. 应当依法报有审批权限的主管部门批准
2. 并报同级保密行政管理部门备案

**不确定时的处理**：
| 情形 | 处理方式 |
|-----|---------|
| 是否属于国家秘密不明确或有争议 | 依法报有关保密行政管理部门确定 |
| 是否属于国家机关工作秘密不明确或有争议 | 报有关业务主管部门确定 |

#### 其他泄露后会对国家安全或公共利益造成不利影响的文件资料

应当按照国家有关规定，严格履行相应程序。

### 三、书面说明要求

**境内企业向有关证券公司、证券服务机构提供文件、资料时**：
1. 应按照国家相关保密规定处理相关文件、资料
2. 就执行保密规定的情况向有关证券公司、证券服务机构提供**书面说明**

> 证券公司、证券服务机构应当妥善保存上述书面说明以备查

### 四、保密协议

经履行相应程序后，向有关证券公司、证券服务机构等提供涉及国家秘密、国家机关工作秘密或其他泄露后会对国家安全或公共利益造成不利影响的文件资料的：

- 双方应当依照《保守国家秘密法》等法律法规及本规定
- 签订**保密协议**
- 明确有关证券公司、证券服务机构等承担的保密义务和责任

### 五、证券公司、证券服务机构义务

| 义务 | 说明 |
|-----|------|
| 保管义务 | 妥善保管获取的文件资料 |
| 系统要求 | 存储、处理、传输上述文件资料的信息系统、信息设备应当符合国家有关规定 |
| 对外提供 | 向境外监管机构和其他相关机构提供、公开披露上述文件资料的，应当按照规定履行相应程序 |

### 六、泄露应急处理

境内企业、证券公司、证券服务机构发现相关文件资料已经泄露或者可能泄露的：

1. 应当立即采取**补救措施**
2. 并及时向有关机关、单位**报告**
3. 机关、单位接到报告后，应当立即作出处理
4. 并及时向保密行政管理等部门报告

### 七、会计档案管理

境内企业向有关证券公司、证券服务机构、境外监管机构等单位和个人提供会计档案或会计档案复制件的：

> 应当按照国家有关规定履行相应程序

### 八、工作底稿管理

| 要点 | 要求 |
|-----|------|
| 存放地点 | 在境内形成的工作底稿应当存放在**境内** |
| 需要出境 | 按照国家有关规定办理**审批手续** |

### 九、监管协作

证监会、财政部、国家保密局和国家档案局等有关主管部门：

- 建立**协作机制**
- 在各自的职权范围内依法对相关事项进行规范和监督检查

### 十、跨境监管合作

**境外证券监督管理机构及有关主管部门**提出就境内企业境外发行上市相关活动进行检查或调查取证的：

| 要求 | 说明 |
|-----|------|
| 检查/调查方式 | 应当通过**跨境监管合作机制**进行 |
| 协助义务 | 证监会或有关主管部门依据双多边合作机制提供必要的协助 |
| 企业配合要求 | 境内有关企业、证券公司、证券服务机构在配合检查、调查或提供文件资料前，应当经**证监会或有关主管部门同意** |

### 十一、法律责任

违反《保守国家秘密法》和《档案法》等法律法规的：
- 由有关部门依法追究**法律责任**
- 涉嫌犯罪的，移送司法机关依法追究**刑事责任**

### 十二、修订要点

本次修订主要内容包括：

| 修订内容 | 说明 |
|---------|------|
| 扩大适用范围 | 将适用范围扩大至境外间接上市企业 |
| 增加程序性要求 | 明确企业保密责任 |
| 明确会计档案管理要求 | 按国家有关规定履行相应程序 |
| 修改境外检查规定 | 删除"现场检查应以我国监管机构为主进行"的表述 |
