# 证监会境外发行上市备案补充材料要求知识库

## 知识库说明

本知识库收录了中国证监会国际合作部发布的**境外发行上市备案补充材料要求**公示信息，旨在为资本市场非诉律师及相关从业人员提供监管关注点的参考。

### 数据来源
- **官网地址**: https://www.csrc.gov.cn/csrc/c100098/common_list.shtml
- **数据范围**: 2023年4月至今的备案补充材料要求公示

```

```

## 常见问题类型解读

### 🔹 股东/实际控制人（占比最高）
监管关注点：
- 5%以上股东的穿透核查
- 股东资格合法性
- 实际控制人认定
- 股权代持还原

### 🔹 股权变动/代持
监管关注点：
- 历次增资转让的定价依据
- 股权代持的形成与解除
- 员工股权激励计划
- 股权变动合法合规性

### 🔹 业务经营/经营范围
监管关注点：
- 经营范围与主营业务匹配性
- 许可经营项目资质
- 业务开展情况
- 行业准入要求

### 🔹 外汇/境外投资
监管关注点：
- 37号文外汇登记
- 境外投资备案
- 红筹架构外汇合规

### 🔹 股权架构/VIE
监管关注点：
- VIE架构搭建合规性
- 协议控制架构拆除
- 返程并购合规性
